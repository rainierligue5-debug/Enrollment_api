# Student Enrollment System

## Quick Start

### Backend (Django)
```powershell
cd c:\PIT-Integ\enrollment_api
python manage.py runserver
```
Backend runs at: **http://localhost:8000**

### Frontend (React)
```powershell
cd c:\PIT-Integ\enrollment_api\frontend
npm start
```
Frontend runs at: **http://localhost:3000**

## What You Get
- **Dashboard** - Overview with statistics
- **Students** - Manage student records
- **Subjects** - Create and manage courses
- **Sections** - Manage class sections with schedules
- **Enrollments** - Enroll students, bulk enrollment, status management

## Requirements
- Python 3.12
- Node.js (npm)
- Django 6.0.2
- React 19.2.4
